PythonFunctions

Python functions are how you do tasks, how you manipulate an object. They are used to preform tasks such as print a value to the screen or convert the data type of a piece of data.


To learn more about functions, go to the Python documentation for built-in functions:
https://docs.python.org/3.13/library/functions.html

(For those of you who don’t know what the Python documentation is, the Python documentation is a place where you can get info about Python. This info is produced by the people who are experts at Python.)

str()

The str() function (also know as the string function) is used for converting non-alphabetical values into alphabetical values. If you had an integer and you wanted it to be a string, you would use the string function. You can tell that the resulting value is a string because it has quotation marks. When something has quotation marks around it, it means that it is a string. Strings cannot have mathematical operations preformed on them but they can be used for other purposes like user input or filling in a sentence. Note: single or double quotes doesn't matter in Python when it comes to strings, they work the same way.

int()

The int() function is the opposite of the str() function. It converts strings to numerical values. For instance, if you have “3” (a string with a value of 3), you can convert to an integer using the int() function. In this example, we converted “3” to 3. Now, 3 is an integer and we can use for many things such as preforming mathematical operations. We can tell that it is an integer because it doesn't have quotes. A value that doesn't have quotes is most of the time a numerical value.

type()

The type() function is used to get the data type of a value. It is especially useful for beginners because it can easily tell you the data type of a value. In image 1, we used the type() function on the string “Hello World!”. The result was str, which is short for string. Therefore, “Hello World!” is a string. In image 2, we also used the type() function on “200”. While it may look like an integer, it has quotations and so it is a string, as the function tells us. Finally, image 3 shows an example of an integer. When we do type(56), we get int because 56 is an integer and there are no quotation marks around it.

help()

The help() function can be used to get info on a function. All you have to do is put help() and then inside the parentheses, put the function such as int or str. In the image above, we received info on the str() function by putting help(str). While the info provided may be difficult for some to understand, after time it will become easier to read.

print()

The print statement is used to print values to the screen. It can take numerical and alphabetical inputs and it has the ability to preform mathematical operations inside of it. Above are various basic ways to implement the print statement.

print(): Basic Strings

In this example, we use a print statement to print the following words. Print statements print output of type string. However, as we will see in future examples, mathematical operations can be preformed inside of them.

print(): f-string

In the image below, we use an f-string to add an extra level of complexity to our statement. To make an f-string, you put print(f" "). Having the f before the quotation mark tells Python that it is an f-string. To better understand the code, lets go step by step. First, the variable “year” is set to an integer with a value of 2023. Next, we use an f-string to input that variable into our print statement. Notice how we put the variable inside of {}. Without the {}, it would literally read “year”. By putting it inside of {}, we can preform mathematical operations and other processing methods that we don't want the viewer to see.

print(): Mathematical Operations

Mathematical operations can also be preformed inside print statements. In the first image, we see that we can preform mathematical operations inside the {} of an f-string. As you see, the {2+2} is calculated and simplified to 4. Then, the output reads “2+2=4”, which is exactly the part that we want the viewer to see. In the second image, it is entirely mathematics and numeric values. There are no strings and so we don't have to tell Python that it is an integer, it just assumes that since there are no quotations, mathematics will be preformed. As intended, our output is only 4, which is exactly what we wanted.
